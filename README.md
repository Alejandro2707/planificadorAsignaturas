
# Planificador de Asignaturas

Universidad Nacional Experimental de Guayana (UNEG)
Carrera: Ingeniería en Informática

## Descripción del Proyecto

Sistema académico de planificación de asignaturas desarrollado en **Java** utilizando **Java Swing** para la interfaz gráfica y una arquitectura modular orientada a objetos inspirada en el patrón **MVC** (Modelo-Vista-Controlador). El sistema procesa y valida las reglas de prelación y disponibilidad de materias del plan de estudios.

---

## Requisitos del Sistema
Para instalar, compilar y ejecutar este software, asegúrese de contar con las siguientes herramientas en su entorno de desarrollo:

* **Java Development Kit (JDK):** Versión 8 o superior (se recomienda JDK 17 o superior).
* **Entorno de Desarrollo (IDE):** Apache NetBeans, IntelliJ IDEA, Eclipse o cualquier IDE compatible con Java.
* **Sistema Operativo:** Windows, macOS o Linux.

---

## Instalación y Configuración

Siga estos pasos detallados para poner en marcha la aplicación en su máquina local:

### 1. Clonar el repositorio u Obtener el Código
Copie la URL del repositorio y ejecute el siguiente comando en su terminal para clonar el proyecto de forma local:
```bash
git clone https://github.com/Alejandro2707/planificadorAsignaturas.git

```

*(Nota: Si descargó el archivo en formato `.zip`, simplemente descomprímalo en una carpeta accesible de su computador).*

### 2. Importar el Proyecto en su IDE

1. Abra su IDE de preferencia (ej. **NetBeans**, **IntelliJ IDEA** o **Eclipse**).
2. Seleccione la opción para abrir o importar un proyecto existente (`Open Project` / `Import Project`).
3. Busque y seleccione la carpeta `planificadorasignaturas` del proyecto.
4. Asegúrese de que el entorno esté configurado para utilizar la versión de **JDK 8 o superior** instalada en su sistema.

### 3. Compilación y Ejecución

1. En el panel de navegación de proyectos de su IDE, localice y seleccione la clase principal de arranque: **`Main.java`**.
2. Ejecute el archivo como una aplicación Java estándar (por ejemplo, haciendo clic derecho y seleccionando `Run File`, o presionando `Shift + F6` en NetBeans).
3. La interfaz gráfica principal de usuario diseñada en Java Swing se desplegará de inmediato dentro del hilo seguro de eventos.
